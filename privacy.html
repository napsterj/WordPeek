WordPeek Privacy Policy

Effective Date: 10th January 2026

Introduction

WordPeek is a Chrome extension that allows users to look up the meaning of a selected word on any webpage using a right-click context menu.

Data Collection

WordPeek does not collect personal or sensitive information. The only data processed is the word selected by the user and its dictionary definitions, which are temporarily stored locally to improve performance.

Data Usage

The selected word and dictionary definitions are used solely to display meanings to the user and to cache recent lookups. No data is used for advertising, analytics, tracking, or profiling.

Data Storage

WordPeek uses Chrome's chrome.storage.sync to temporarily store selected words and their definitions. This data is non-personal and is synced only across the userâ€™s devices via Chrome.

Data Sharing

WordPeek does not sell, share, or transfer user data to third parties. Selected words are sent only to the dictionary API to fetch definitions.

Permissions

  • contextMenus: Required to add a right-click menu option for looking up words.
  • host permissions: Required to fetch word definitions from a dictionary API.
  • storage: Required to temporarily store selected words and their definitions for caching purposes.

User Control

WordPeek only acts when the user selects a word and clicks the context menu. No background tracking occurs.
